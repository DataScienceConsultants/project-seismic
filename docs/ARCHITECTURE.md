# EQ Tracker Architecture

EQ Tracker is built as a modular vanilla JavaScript application.

## Main Layers

### API
External data sources such as USGS earthquake feeds.

### Map
Leaflet initialization, map layers, markers, and legends.

### UI
Status cards, earthquake cards, bottom sheets, and screen elements.

### Services
Browser services such as geolocation, storage, and future notifications.

### Utils
Shared helper functions.
