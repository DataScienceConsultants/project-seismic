# EQ Tracker Design System

## Design Philosophy

Quiet Confidence.

The interface should feel calm, clear, premium, and trustworthy.

## Visual Direction

- Dark interface with vivid earthquake accents.
- Neutral UI colors.
- Earthquake colors communicate severity.
- Large typography.
- Spacious layout.
- Mobile-first.
- Map as the main visual canvas.

## Components

Core reusable components:
- Status card
- Earthquake card
- Bottom sheet
- Alert badge
- Primary button
- Secondary button
- Map overlay
- Notification preview
